## 合并页面

中英文左右对照合并

## 一、两份文档，英文.pdf，中文.pdf合为一份

1.先备份英文.pdf，然后打开

2.组织，调整大小

![image-20260428144146934](./ReadMe.assets/image-20260428144146934.png)

3.设置范围，全部，百分比取消关联，并且选200%：100%，左对齐

![image-20260428144235969](./ReadMe.assets/image-20260428144235969.png)

4.确定即可，这里目的是把英文文档扩大一倍，右边让中文页面来覆盖。

5.组织，覆盖

![image-20260428144401477](./ReadMe.assets/image-20260428144401477.png)

6.选择中文.pdf，水平对齐，右边

![image-20260428144518673](./ReadMe.assets/image-20260428144518673.png)

7.确定即可。
